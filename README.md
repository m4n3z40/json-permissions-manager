# JSON Permissions Manager

A desktop app to manage JSON Permission files. You probably don't need this.

Built with [Squark.](https://github.com/m4n3z40/Squark)

### Development

To run tslint:
```sh
npm run lint
```

To build the source:
```sh
npm run build
```

To run the app:
```sh
npm run app
```

To start watchers:
```sh
npm run watch
```

To start watchers and run app:
```sh
npm start
```
